最近，中国有关垃圾处理的讨论中，出现了一句听来颇为奇怪的话：

```text
垃圾不够烧了。
```

乍听之下，几乎不可思议。

多年来，中国城市面对的明明是相反的问题：垃圾太多，无处安放，人们担心的是“垃圾围城”。

那么，一个国家怎么会从垃圾太多，走到垃圾不够烧？

答案并不是垃圾消失了。

改变的是整个垃圾处理体系。

过去十年，许多中国城市以惊人的速度扩建垃圾焚烧发电设施。在一些地方，如今的问题已经不再是城市能否处理每天产生的生活垃圾，而是焚烧厂能否获得足够的垃圾，按原先设计的规模运转。

这正是“垃圾不够烧”值得琢磨的地方。

它说的不只是垃圾。

这句话意味着，一座城市的垃圾管理已经从一个阶段迈入另一个阶段。

## 焚烧的首要目的并不是发电

人们听到“垃圾焚烧发电”，很容易以为重点在发电。

发电当然是其中一环，却不是城市选择焚烧最根本的原因。

真正迫使城市走向焚烧的，是土地压力。

在人口密集的城市，传统填埋很难长久维持。垃圾填埋场不仅长期占用土地，还会产生必须谨慎处理的渗滤液、释放填埋气体。即使封场，往往仍需持续监测和维护几十年。

焚烧改变了问题的形态。

城市不再需要把大量混合垃圾直接埋入地下，而是可以在受控设施中焚烧可燃垃圾，回收部分能量，大幅缩减垃圾体积，再将剩余物集中为灰渣和其他残余物。

按照新加坡官方的说法，焚烧可以使垃圾体积减少约90%。[^nea-wte]

这个数字是关键。

焚烧的价值，并不在于让垃圾凭空消失。

它只是把问题压缩了。

```text
大量日常垃圾
  -> 少得多的灰渣和残余物
```

但压缩不等于消失。

烧完以后，仍然会有东西留下。

也正是从这里开始，新加坡的经验显得格外有意思。

## 新加坡早已进入焚烧之后的阶段

新加坡是一个很有代表性的例子，因为对它而言，土地从来不是取之不尽的缓冲空间。

国土面积小，人口密度高，经济活动又高度集中于城市。如果主要依赖大规模填埋，垃圾处理就会直接与住宅、工业、港口、自然环境、水资源管理及其他土地用途争夺空间。

因此，新加坡把大部分可燃垃圾送往垃圾焚烧发电厂，并从焚烧底渣中回收金属。剩余灰渣再与不可焚烧垃圾一起运往海上的实马高垃圾埋置场（Semakau Landfill）。[^nea-semakau]

这一点会彻底改变人们对垃圾去向的理解。

对居民来说，垃圾一旦投入垃圾槽或送到垃圾收集站，似乎就已经处理完了。

对城市而言，处理才刚刚开始。

```text
住宅 / 商店 / 办公室
  -> 垃圾收集站或收集点
  -> 垃圾运输车
  -> 垃圾焚烧发电厂
  -> 灰渣和残余物
  -> 大士海上转运站
  -> 实马高垃圾埋置场
```

一只普通的垃圾桶，不过是国家级物流体系面向公众的一扇前门。

![新加坡并排放置的一只普通垃圾桶和一只蓝色回收箱](assets/when-cities-run-out-of-garbage-to-burn/06-general-and-recycling-bins.jpg)

*两只普通的桶，通往两条不同的路线。一般垃圾与干净可回收物在这里分流，这个小小的动作，正是整套城市系统的起点。*

## 城市运送的不只是垃圾

要概括其中的差别，很容易写出这样一句醒目的话：

```text
有些城市运输垃圾。
新加坡运输灰渣。
```

这句话很好记，却不能照字面理解。

新加坡每天依然要收集和运输普通垃圾。住宅、商场、食阁、办公室和公共空间产生的垃圾，仍要由卡车运走。

真正的区别，在于最后送进填埋场的究竟是什么。

在以填埋为主的体系中，更多未经处理的原始垃圾会被直接埋入地下。

而在新加坡，最终进入填埋场的，主要是焚烧后较少的剩余物：灰渣、其他残余物，以及不可焚烧垃圾。

问题已经变了。

城市不再只问：

```text
这么多垃圾，该放到哪里？
```

它开始问：

```text
焚烧已经大幅减少了垃圾，
剩下的东西又该怎么办？
```

这就是垃圾管理的下一个阶段。

## 实马高不只是填埋场，也是一场倒计时

实马高垃圾埋置场很容易让人产生误解。

从远处看，人们也许会把它想象成一座不断扩大的原始垃圾岛。

但实马高并不是把松散的生活垃圾直接倒进海里。它是一座经过工程设计的海上垃圾填埋场。焚烧产生的灰渣由船只运到那里，再填入设有围护设施的区域。

与许多人印象中的填埋场相比，这种处置方式更清洁，也更可控。

但容量问题并不会因此消失。

实马高的容量仍然有限。

每运入一吨灰渣，新加坡仅有的最终填埋空间就会少一分。即使焚烧能让垃圾体积减少90%，在土地如此稀缺的地方，剩下的10%仍不可忽视。

因此，实马高不仅是垃圾处理链条的终点。

它也是一只时钟。

只要城市继续产生垃圾，填埋场就会一天天接近满载。

![从大士垃圾处理设施到实马高垃圾埋置场的简化路线](assets/when-cities-run-out-of-garbage-to-burn/04-tuas-semakau-waste-route.svg)

*这是路线示意图，并非按比例绘制的地图。重要的是先后顺序：先在大士焚烧，再经海上转运，最后送往实马高处置。*

## 未来不是烧得更多

当一座城市已经擅长焚烧垃圾时，接下来的问题就不只是如何建造更多焚烧厂。

更难的问题是，怎样减少焚烧后留下的东西。

新加坡未来的垃圾处理策略，正朝这个方向发展。

NEWSand就是一例。新加坡正在研究，经过处理的焚烧底渣和其他垃圾残余物能否转化为建筑材料，用于道路或非结构性工程。[^nea-newsand]

思路并不复杂，却十分重要：

```text
如果灰渣能够安全地变成材料，
需要送去填埋的灰渣就会减少。
```

当然，这并不意味着所有残余物都可以随意再利用。安全标准不可或缺，重金属、浸出风险和长期环境影响都必须得到控制。

但方向已经很明确。

下一阶段的任务，不只是处理垃圾，更是回收材料。

城市开始追问：今天被当成“废物”的东西，能不能重新成为可用的原料？

## Tuas Nexus：把垃圾、水和能源放进同一套系统

新加坡未来的一部分，是Tuas Nexus。

Tuas Nexus将综合废物管理设施（Integrated Waste Management Facility）与大士水回收厂（Tuas Water Reclamation Plant）建在同一地点。垃圾处理与用过的水处理从一开始就统筹规划，两套设施之间的能源和资源因而可以相互利用。[^nea-iwmf]

这很重要，因为对一座城市来说，“垃圾”“水”和“能源”从来不是三个互不相干的问题。

它们共同构成了城市的新陈代谢。

食物最终变成垃圾。

用过的水经过处理，会留下污泥。

垃圾可以转化为能源。

回收的材料可以重新用于建筑业或工业。

这些物质流衔接得越紧密，就越不必把每一个环节都当成孤立的问题来解决。

这也让我们换了一种眼光看待基础设施。

一座座处理厂并非各自为政，而更像城市体内相互连接的器官。

![新加坡从日常垃圾到焚烧、回收和实马高处置的系统流程](assets/when-cities-run-out-of-garbage-to-burn/05-singapore-waste-system-flow.svg)

*垃圾桶只是第一个接口。接下来，城市还要判断哪些东西可以焚烧、回收或再利用，哪些最终必须送往实马高。*

## 回收才是日常生活中更难的一环

垃圾桶之所以重要，还有一个原因。

大型处理设施可以靠工程建起来，可回收物是否合格，却要靠每个人在日常生活中把关。

新加坡许多社区都设有蓝色回收箱。它们随处可见，人们再熟悉不过，也很容易熟视无睹。

但真正的问题，不在于有没有回收箱。

而在于人们往里面扔了什么。

![印有纸张、金属、塑料和玻璃回收说明的蓝色回收箱](assets/when-cities-run-out-of-garbage-to-burn/01-recycling-bin-close-up.jpg)

*这张说明标签要交代的事情不少：只收可回收物，不能放食物和液体，投入前还要冲洗干净。*

可回收物一旦沾上食物或液体，或是混进不可回收物，后续处理就会困难得多，甚至根本无法回收。一个回收箱从外面看或许象征着进步，里面收集的材料却未必有多少利用价值。

所以，垃圾管理的未来，不只取决于大士的大型设施。

也取决于居民下楼扔垃圾时所做的一个个小判断：

```text
这个瓶子倒空了吗？
这个容器洗干净了吗？
这件东西真的能回收吗？
它该放进蓝色回收箱，还是普通垃圾桶？
```

城市可以建起一整套回收系统。

但最终进入这套系统的是什么，仍由居民决定。

垃圾桶上的标签，也是一种微型公共基础设施。它要把一套全国性的回收体系，变成普通人五秒钟内就能做出的几个选择。

## “垃圾不够烧”究竟意味着什么

那么，“垃圾不够烧了”究竟说明了什么？

它说明，垃圾管理会经历不同阶段。

第一阶段，是生存：

```text
把垃圾收走。
让街道保持清洁。
不让垃圾越积越多。
```

第二阶段，是把垃圾纳入可控的处理体系：

```text
告别失控的随意倾倒。
减少对填埋场的依赖。
通过焚烧或其他体系安全处理垃圾。
```

第三阶段，是回收资源：

```text
回收金属。
回收能源。
回收材料。
减少最终残余物。
保护最后的填埋容量。
```

中国出现“垃圾不够烧”的现象之所以值得关注，是因为它表明，一套体系可以迅速从处理能力不足，走到部分地区可能出现焚烧能力过剩。

新加坡值得关注，则是因为它早已开始面对焚烧之后的问题：

```text
烧完以后剩下的东西，该怎么办？
```

## 垃圾桶背后的隐形系统

在认真思考这个问题之前，我一直把垃圾桶当成处理过程的终点。

把东西扔进去。

盖上盖子。

它就消失了。

但对城市来说，世上并没有一个真正可以把东西“扔掉”的地方。

垃圾不过是被送往系统的下一站。

垃圾桶连着卡车。

卡车驶向处理厂。

处理厂留下灰渣。

灰渣再被运往实马高。

而实马高最终把我们引向一个更大的问题：

```text
只要人们还在生活、吃饭、购物，还在不断丢弃东西，
城市能不能让最终产生的废物越来越少？
```

这正是垃圾管理值得放进城市系列里讨论的原因。

一切都从每天路过、平凡得让人几乎不会多看一眼的东西开始。

然后一步步显露出那套维持高密度城市运转的隐形机器。

越是观察新加坡，我越觉得，这里最有意思的系统，往往并不是最显眼的那些。

有时，线索始于一个树桩。

有时，则始于一只垃圾桶。
