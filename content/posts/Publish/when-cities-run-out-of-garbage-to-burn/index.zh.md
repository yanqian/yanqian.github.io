最近，中国的垃圾处理讨论中出现了一句听来有些古怪的话：

```text
垃圾不够烧了。
```

乍听之下，几乎不可思议。

多年来，人们谈起中国城市的垃圾问题，想到的明明是另一番景象：垃圾太多，无处安放，还要担心“垃圾围城”。

一个曾经垃圾过剩的国家，怎么会变成垃圾不够烧了？

答案并不是垃圾消失了。

变的是垃圾处理系统。

过去十年，许多中国城市以惊人的速度扩建垃圾焚烧发电设施。于是，在一些地方，问题已不再是城市能否处理每天产生的生活垃圾，而是焚烧厂能否收到足够的垃圾，按设计规模运转。

这正是那句话耐人寻味之处。

它说的不只是垃圾够不够烧。

它意味着，一座城市的垃圾管理已经跨入新的阶段。

## 焚烧首先解决的并不是发电

听到“垃圾焚烧发电”，人们很容易把重点放在发电上。

发电当然是其中一环，却不是采用焚烧的根本原因。

真正促使高密度城市选择焚烧的，是土地压力。

在人口密集的城市，传统填埋场很难长期维持。它会常年占用土地，还会产生必须谨慎处理的渗滤液和填埋气体。即使关闭以后，往往仍需接受数十年的监测和维护。

焚烧改变了问题的形态。

城市不再把大量混合垃圾直接埋入地下，而是在受控设施中焚烧可燃垃圾，回收一部分能源，同时大幅压缩垃圾体积，最后把剩余物集中为灰渣和其他残余物。

按照新加坡官方的说法，焚烧可以使垃圾体积减少约90%。[^nea-wte]

这个数字是关键。

焚烧的价值，不在于让垃圾凭空消失。

它只是压缩了问题。

```text
大量日常垃圾
  -> 少得多的灰渣和残余物
```

但压缩并不等于消失。

烧完以后，仍有东西留下。

也正是从这里开始，新加坡的例子变得格外有意思。

## 新加坡早已进入下一个阶段

新加坡很适合用来观察这个问题，因为它无法把土地当作取之不尽的缓冲区。

这里国土狭小、人口密集，经济和生活又高度城市化。如果建立一套以大规模填埋为主的系统，垃圾处理就会直接与住宅、工业、港口、自然保护、水资源管理及其他土地用途争夺空间。

因此，新加坡把大部分可燃垃圾送往垃圾焚烧发电厂。焚烧底渣中的金属可以回收，剩余灰渣则和不可焚烧垃圾一起运往新加坡的海上填埋场——实马高垃圾埋置场。[^nea-semakau]

这套流程也改变了我们理解垃圾去向的方式。

对居民来说，把垃圾投入垃圾槽或送到垃圾收集站，事情似乎就结束了。

对城市而言，一切才刚刚开始。

```text
住宅 / 商店 / 办公室
  -> 垃圾收集站或收集点
  -> 垃圾收运车
  -> 垃圾焚烧发电厂
  -> 灰渣和残余物
  -> 大士海上转运站
  -> 实马高垃圾埋置场
```

一个普通的垃圾桶，其实是整套全国物流系统的入口。

![新加坡的一只普通垃圾桶与一只蓝色回收桶并排摆放](assets/when-cities-run-out-of-garbage-to-burn/06-general-and-recycling-bins.jpg)

*两个普通的桶，通向两条不同的路径。庞大的城市系统，就从普通垃圾与洁净可回收物的这次小小分流开始。*

## 城市运送的不只是垃圾

要概括其中的差别，有句话很容易让人记住：

```text
有些城市运送垃圾。
新加坡运送灰渣。
```

这句话很醒目，却不能只看字面。

新加坡当然仍在收集和运输日常垃圾。住宅、商场、食阁、办公室和公共空间产生的废弃物，依旧要由卡车运走。

真正的区别在于，最终进入填埋场的究竟是什么。

在以填埋为主的系统中，更多未经处理的原始垃圾会被直接埋入地下。

而在新加坡，最终送进填埋场的主要是焚烧后大幅减量的剩余物：灰渣、其他残余物和不可焚烧垃圾。

问题的性质已经不同了。

城市问的不再只是：

```text
这么多垃圾该放到哪里？
```

它还要追问：

```text
用焚烧减少垃圾以后，
剩下的东西该怎么办？
```

这就是垃圾管理的下一个阶段。

## 实马高不只是一座填埋场，也是一场倒计时

实马高垃圾埋置场很容易被误解。

若只凭远处的想象，人们或许会以为，那是一座由未经处理的垃圾不断堆成的岛。

但实马高并不是把散乱的生活垃圾直接倒进海里。它是一座经过工程设计的海上填埋场。焚烧产生的灰渣经海运抵达后，再填入设有围护设施的区域。

与许多人印象中的填埋场相比，这里更洁净，管理也更严密。

但工程手段无法抹去容量上限。

实马高终究会装满。

每运入一吨灰渣，就会消耗新加坡一部分最终填埋容量。即使焚烧能让垃圾体积减少90%，对土地稀缺的地方来说，剩下的10%依然不容忽视。

所以，实马高不只是垃圾处理系统的终点。

它也是一只钟。

只要城市还在产生垃圾，填埋场就会不断趋近满载的那一刻。

![从大士垃圾处理设施到实马高垃圾埋置场的简化路线](assets/when-cities-run-out-of-garbage-to-burn/04-tuas-semakau-waste-route.svg)

*这是一张简化的路线示意图，并非按比例绘制。重要的是处理顺序：先在大士焚烧，再经海上转运，最后送到实马高处置。*

## 未来不是烧得更多

一座城市一旦建成高效的焚烧体系，接下来就不该只问怎样再建更多焚烧厂。

更难的问题是：怎样减少焚烧后留下的东西？

新加坡未来的垃圾处理策略，正朝这个方向推进。

NEWSand就是一个例子。新加坡正在研究，经过处理的焚烧底渣和其他废弃残余物，能否用作道路等非结构用途的建筑材料。[^nea-newsand]

这个思路简单，却很重要：

```text
如果灰渣能安全地转化为材料，
需要送去填埋的灰渣就会减少。
```

当然，这并不意味着任何残余物都可以随意再利用。安全标准至关重要，重金属、浸出风险和长期环境影响都必须得到控制。

但方向已经很清楚。

下一阶段不能止步于处理垃圾。

还要回收材料。

城市开始追问：今天被视作“垃圾”的东西，能否重新转化为可用的投入？

## 大士综合设施：把垃圾、水与能源纳入同一套系统

新加坡未来规划的另一部分，是大士综合设施（Tuas Nexus）。

它把综合废物管理设施与大士水回收厂结合在一起。核心思路是共址建设：把垃圾处理和用过水处理放在同一处统筹规划，让能源流和资源流相互支撑。[^nea-iwmf]

这很重要，因为一座城市面对的，并不是几个互不相干、分别名为“垃圾”“水”和“能源”的问题。

它们共同构成了城市代谢。

食物变成垃圾。

用过的水产生污泥。

垃圾可以转化为能源。

回收材料又可以重新流入建筑业或工业。

系统整合得越深入，各种物质流就越不会被当成孤立的问题来处理。

基础设施也因此显露出另一种面貌。

它们不再是一座座彼此隔绝的工厂，更像城市体内相互连接的器官。

![新加坡垃圾从日常废弃物进入焚烧、资源回收并最终抵达实马高的流程](assets/when-cities-run-out-of-garbage-to-burn/05-singapore-waste-system-flow.svg)

*垃圾桶只是第一个接口。此后，城市系统会决定哪些东西可以焚烧、回收或再利用，哪些最终必须送往实马高。*

## 回收才是日常生活中更难的一环

垃圾桶之所以重要，还有另一个原因。

工业处理系统可以靠工程建设起来，但回收物的质量，首先取决于每个人日常如何投放。

新加坡许多社区都设有蓝色回收桶。它们随处可见，人人熟悉，也因此很容易被忽略。

然而，真正重要的并不是有没有回收桶。

而是人们往里面扔了什么。

![蓝色回收桶上的说明涵盖纸张、金属、塑料和玻璃](assets/when-cities-run-out-of-garbage-to-burn/01-recycling-bin-close-up.jpg)

*这张说明标签要交代的事情不少：只能投入可回收物，不能残留食物和液体，投放前还要冲洗干净。*

一旦混入食物、液体或不可回收物，这些材料就会更难回收，甚至无法回收。一个回收桶从外面看像是进步的象征，里面产出的材料却仍可能品质低下。

所以，垃圾管理的未来不只取决于大士的大型设施。

也取决于居民下楼扔垃圾时所做的几个小判断：

```text
瓶子倒空了吗？
容器洗干净了吗？
这件东西真的能回收吗？
应该扔进蓝色回收桶，还是普通垃圾桶？
```

城市可以把整套系统建起来。

但送进系统的究竟是什么，仍由居民决定。

垃圾桶上的标签，也是一种微型公共基础设施。它要把一整套全国回收体系，变成几项让人能在五秒钟内做出的选择。

## “垃圾不够烧”究竟意味着什么

那么，“垃圾不够烧”到底告诉了我们什么？

它说明，垃圾管理会经历不同的发展阶段。

第一阶段，是维持基本运转：

```text
把垃圾及时收走。
让街道保持清洁。
不让垃圾越积越多。
```

第二阶段，是受控处理：

```text
告别不受控制的倾倒。
减少对填埋场的依赖。
通过焚烧或其他系统安全处理垃圾。
```

第三阶段，是资源回收：

```text
回收金属。
回收能源。
回收材料。
减少最终残余物。
保护最后一座填埋场的容量。
```

中国出现“垃圾不够烧”的现象，值得关注。它表明，一套系统可以在很短时间内从处理能力不足，走到部分地区可能产能过剩的阶段。

新加坡之所以值得观察，是因为它早已开始面对焚烧之后的问题：

```text
剩下的东西，该怎么办？
```

## 垃圾桶背后的隐形系统

在认真思考这个问题之前，我一直把垃圾桶当作一段流程的终点。

把东西扔进去。

盖上盖子。

它就消失了。

但对一座城市而言，并不存在一个真正可以把东西扔过去的“别处”。

垃圾只是被移交给系统的下一个环节。

垃圾桶连着收运车。

收运车驶向处理厂。

处理厂留下灰渣。

灰渣再被送往实马高。

而实马高最终把我们带到一个更大的问题面前：

```text
当人们继续生活、吃饭、购物，也继续丢弃东西时，
一座城市能否让最终需要处置的垃圾越来越少？
```

这正是垃圾管理值得写进城市系列的原因。

故事从一件再普通不过、让人每天经过也不会多看一眼的东西开始。

顺着它看下去，一套隐形而庞大的机器逐渐显露出来。正是这套系统，支撑着高密度城市的日常运转。

我越是观察新加坡，越觉得这里最有意思的系统，往往并不是最显眼的那些。

有时，线索始于一截树桩。

有时，则始于一只垃圾桶。
