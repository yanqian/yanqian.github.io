最近，中国关于垃圾处理的讨论中，出现了一句听来颇为奇怪的话：

```text
没有足够的垃圾可烧了。
```

乍听之下，这几乎不可能。

多年来，人们谈起中国城市的垃圾问题，想到的明明是另一番景象：垃圾太多，无处安放，甚至担心“垃圾围城”。

那么，一个国家怎么会从垃圾太多，走到垃圾不够烧？

答案并不是垃圾消失了。

变的是垃圾处理系统。

过去十年，许多中国城市以惊人的速度扩建垃圾焚烧发电设施。如今，有些地方的问题已不再是城市产生的生活垃圾能否得到处理，而是焚烧厂能否获得足够的垃圾，按设计规模运行。

这句话之所以值得注意，原因就在这里。

它说的不只是垃圾。

它意味着，一座城市的垃圾管理已经从一个阶段迈入另一个阶段。

## 焚烧的首要意义并不是发电

一听到“垃圾焚烧发电”，人们很容易以为重点在发电。

发电当然是其中一环，却不是最深层的一环。

真正迫使城市走向焚烧的，是土地压力。

在人口密集的城市，传统填埋场很难长期维持。填埋不仅会长期占用土地，还会产生必须谨慎处理的渗滤液，并释放填埋气体。即使封场之后，往往仍需持续监测和维护数十年。

焚烧改变了问题的形态。

城市不再把大量混合垃圾直接埋入地下，而是在受控设施中焚烧可燃垃圾，回收部分能源，大幅缩小垃圾体积，再将余下的物质集中为灰渣和其他残余物。

按照新加坡官方的说法，焚烧可以使垃圾体积减少约90%。[^nea-wte]

这个数字至关重要。

焚烧的价值，不在于让垃圾凭空消失。

它只是把问题压缩了。

```text
大量日常垃圾 -> 少得多的灰渣和残余物
```

但压缩不等于消失。

焚烧之后，仍然会有东西留下。

新加坡的意义，也正是从这里显现出来。

## 新加坡早已进入下一个阶段

新加坡是一个很有代表性的例子，因为它不可能把土地当成取之不尽的缓冲地带。

这个国家面积小、人口密集，经济高度城市化。如果主要依靠大规模填埋，垃圾处理便会直接同住房、工业、港口、自然环境和水资源管理争夺土地。

因此，新加坡建立了一套以焚烧为主的系统：大部分可燃垃圾进入垃圾焚烧发电厂；可以从焚烧底灰中回收金属；剩余灰渣则与不可焚烧垃圾一起，送往新加坡的海上填埋场——实马高填埋场。[^nea-semakau]

这套流程会改变我们看待垃圾的方式。

对居民来说，垃圾扔进垃圾槽或集中收集点，事情似乎就结束了。

对城市而言，一切才刚刚开始。

```text
住宅 / 商店 / 办公室
  -> 垃圾集中收集点或其他收集点
  -> 垃圾清运车
  -> 垃圾焚烧发电厂
  -> 灰渣和残余物
  -> 大士海运转运站
  -> 实马高填埋场
```

一个普通垃圾桶，不过是整个国家物流系统的入口。

![新加坡一个普通垃圾桶与一个蓝色回收桶并排摆放](assets/when-cities-run-out-of-garbage-to-burn/06-general-and-recycling-bins.jpg)

*两个普通的桶，通向两条不同的路线。一般垃圾与洁净可回收物之间这道小小的分流，正是城市级系统的起点。*

## 城市运送的不只有垃圾

如果想用一句话概括这种差别，很容易写成：

```text
有些城市运送垃圾。新加坡运送灰渣。
```

这句话很好记，却不能照字面理解。

新加坡依然每天收集和运输垃圾。清运车照样要从住宅、商场、美食中心、办公室和公共空间运走废弃物。

真正的差别，在于最后进入填埋场的究竟是什么。

在以填埋为主的系统中，大量未经处理的原始垃圾会被直接埋入地下。

而在新加坡，最终填埋的主要是焚烧后体积大幅缩小的余留物：灰渣、其他残余物，以及不可焚烧垃圾。

问题已经截然不同。

城市不再只问：

```text
这么多垃圾要放到哪里？
```

它开始问：

```text
用焚烧减少垃圾之后，剩下的东西该怎么办？
```

这就是垃圾管理的下一个阶段。

## 实马高不只是一座填埋场，也是一场倒计时

人们很容易误解实马高填埋场。

隔着距离想象它，脑海中或许会浮现一座由原生垃圾堆成、不断扩大的岛。

但实马高并不是把散装生活垃圾直接倒进海里。它是一座经过工程设计的海上填埋场。焚烧后的废弃物经海路运抵，再填入设有围护设施的区域。

因此，它比人们通常想象的填埋场更整洁，也更可控。

但容量问题并不会因此消失。

实马高终究有装满的一天。

每运入一吨灰渣，新加坡仅存的填埋容量就会再少一分。即使焚烧能使垃圾体积减少90%，在土地稀缺的地方，余下的10%仍然不容忽视。

所以，实马高不仅是垃圾处理系统的终点。

它也是一只时钟。

只要城市还在产生垃圾，填埋场就会一步步接近满载。

![从大士垃圾处理设施到实马高填埋场的简化路线](assets/when-cities-run-out-of-garbage-to-burn/04-tuas-semakau-waste-route.svg)

*这是一张路线概念图，并非按比例绘制的地图。重要的是先后顺序：垃圾在大士焚烧，经海上转运，最后送往实马高处置。*

## 未来不是继续增加焚烧

当一座城市已经能够有效焚烧垃圾，接下来要问的，就不该只是怎样再建更多焚烧厂。

更难的问题是：怎样减少焚烧后留下的东西？

新加坡未来的垃圾管理策略，正朝着这个方向推进。

NEWSand就是一个例子。新加坡正在探索，经过处理的焚烧底灰和其他废弃物残余，能否用作道路或非结构性工程等领域的建筑材料。[^nea-newsand]

其中的道理简单，却很重要：

```text
如果灰渣能够安全地变成材料，需要送去填埋的灰渣就会更少。
```

这并不意味着所有残余物都可以随意再利用。安全标准不可或缺，重金属、浸出风险和长期环境影响都必须得到控制。

但方向已经很清楚。

下一阶段不能只有垃圾处理。

还要回收材料。

城市开始追问：眼下被归为“垃圾”的东西，能不能重新成为有用的生产原料？

## 大士综合设施：把垃圾、水和能源纳入一套系统

新加坡未来布局的另一部分，是大士综合设施（Tuas Nexus）。

大士综合设施将综合废物管理设施与大士水回收厂结合起来。关键在于两者共址建设、统一规划：垃圾管理与用过的水处理不再各自为政，能源和资源可以在两个系统之间流动，相互支撑。[^nea-iwmf]

因为对一座城市而言，所谓“垃圾”“水”和“能源”，从来不是三个互不相干的问题。

城市只有一套新陈代谢。

食物会变成垃圾。

用过的水会产生污泥。

垃圾可以转化为能源。

回收材料可以重新进入建筑业或工业。

系统整合得越深入，各类物质就越不必被视为彼此孤立的问题，分头处置。

这也为我们理解基础设施提供了另一种视角。

我们看到的不再是一座座互不相干的处理厂，而是城市体内彼此相连的器官。

![新加坡从日常垃圾到焚烧、资源回收和实马高处置的垃圾系统流程](assets/when-cities-run-out-of-garbage-to-burn/05-singapore-waste-system-flow.svg)

*垃圾桶只是第一个接口。此后，城市系统会判断哪些东西可以焚烧、回收、再利用，哪些最终必须送往实马高。*

## 回收才是日常生活中更难的一环

垃圾桶之所以重要，还有一个原因。

工业设施可以精心设计，回收物干不干净、能不能用，却要从每个人日常投放的那一刻说起。

新加坡许多社区都设有蓝色回收桶。它们随处可见，大家再熟悉不过，也很容易熟视无睹。

但真正的问题不在于有没有回收桶。

而在于人们往里面放了什么。

![一个蓝色回收桶，上面标有纸张、金属、塑料和玻璃的投放说明](assets/when-cities-run-out-of-garbage-to-burn/01-recycling-bin-close-up.jpg)

*这张投放标签要交代的事情不少：这里只收可回收物，不能带有食物残渣和液体，投放前还得冲洗干净。*

如果可回收物沾有食物或液体，或是混入不可回收物，后续处理就会困难得多，甚至失去回收价值。一个回收桶从外面看像是进步的象征，最终收上来的却可能只是一批品质低劣的材料。

因此，垃圾管理的未来，不只取决于大士的那些大型设施。

它也取决于人们下楼后做出的一个个小判断：

```text
这个瓶子倒空了吗？
这个容器洗干净了吗？
这件东西真的能回收吗？
它该放进蓝色回收桶，还是普通垃圾桶？
```

城市可以建起一套回收系统。

但究竟有什么进入这套系统，仍由居民决定。

垃圾桶上的标签，也是一小块公共基础设施。它要把一整套国家回收体系，化成每个人五秒钟左右就能做完的几道选择题。

## “没有足够的垃圾可烧”究竟意味着什么

那么，“没有足够的垃圾可烧”究竟说明了什么？

它意味着，垃圾管理也有不同的发展阶段。

第一阶段，是先让城市正常运转：

```text
把垃圾收走。
让街道保持整洁。
不让垃圾越积越多。
```

第二阶段，是把垃圾纳入受控处理：

```text
告别不受控制的倾倒。
减少对填埋场的依赖。
通过焚烧或其他系统安全处理垃圾。
```

第三阶段，才是资源回收：

```text
回收金属。
回收能源。
回收材料。
减少最终残余物。
保护所剩不多的填埋容量。
```

中国出现“没有足够的垃圾可烧”的现象之所以值得关注，是因为它表明，一些地方可能在很短的时间里，便从处理能力不足走到了产能可能过剩的另一端。

新加坡值得观察，则是因为它早已开始面对焚烧完成之后的问题：

```text
剩下的东西该怎么办？
```

## 垃圾桶背后的隐形系统

在认真思考这个问题之前，我一直把垃圾桶当成一段流程的终点。

你扔掉一样东西。

桶盖合上。

它就不见了。

可在城市里，并没有什么东西会真正被“扔掉”。

它只是被送到了系统中的另一个地方。

垃圾桶连着清运车。

清运车驶向处理厂。

处理厂留下灰渣。

灰渣再被送往实马高。

实马高则把我们带到一个更大的问题面前：

```text
只要人们还在生活、吃饭、购物和丢弃物品，一座城市能否继续减少最终需要处置的垃圾？
```

所以，垃圾管理很适合写进这个城市系列。

它从一件再普通不过、让人每天路过也不会多看一眼的东西讲起。

随后，一点点揭开支撑高密度城市运转的那套隐形机器。

观察新加坡越久，我越觉得，这里最有意思的系统，往往不是最显眼的那些。

有时，线索始于一个树桩。

有时，则始于一个垃圾桶。
