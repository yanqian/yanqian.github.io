# Obsidian Publishing

- Obsidian is the private source of truth.
- Only notes marked `public: true` should be published.
- Published notes should go to `content/posts/`.
- Do not publish to the repository root.
- Avoid publishing private notes, work details, secrets, or raw personal notes.
- Obsidian wikilinks like `[[Kafka]]` should be converted or avoided in public posts.
